# Aquarium World

Aquarium World is a lightweight aquarium simulator designed for low-end PCs and Android devices.

## Vision

Everyone can build their dream aquarium.

## Focus

- Performance first
- Low-end PC and Android support
- Animated aquarium, not only tables
- Fish AI
- Water quality
- Feeding
- Cleaning
- Shop
- Reputation progression

## Current Goal

Build Alpha v0.1 as a playable HTML5 prototype.
