# Aquarium World Roadmap

## Alpha v0.1 – Playable Aquarium Prototype

Goal: create the first playable version with visible aquarium gameplay.

### Must Have

- Visual aquarium scene
- Animated fish movement
- Plants, rocks, sand and bubbles
- Feed button with food animation
- Clean button
- Water Change button
- Coins system
- Reputation system
- Water quality, hunger, oxygen and ammonia
- Save and reset system

### Performance Target

- Runs smoothly on weak laptops
- Works in Android browser
- No heavy frameworks
- Simple HTML, CSS and JavaScript

## Alpha v0.2 – Living Aquarium

- Fish behavior by species
- Better feeding reaction
- Basic shop
- Inventory
- More fish types
- Plant decorations

## Alpha v0.3 – Hybrid Progression

- Daily quests
- Reputation ranks
- Visitor coins
- Tank upgrades
- Filter and light upgrades

## Future Unity Version

After the HTML5 prototype is stable, we will migrate the core gameplay to Unity for Android and Windows builds.
